# SKYMUNDO Luxury Website Refresh

This package includes a modern one-page luxury travel site with:

- corrected logo colors (`SKY` blue / `MUNDO` orange)
- footer logo presented on white over dark blue background
- improved mobile layout
- corrected destination imagery
- upgraded premium visual style

## Files

- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.svg`
- `assets/images/*`

## Before going live

- `hello@skymundo.ae`
- `+971000000000`
- `https://wa.me/971000000000`
